# GEIE - Prototipo de Gestao de Espacos e Infraestrutura Escolar

Prototipo de baixa fidelidade (apenas para apresentacao) da atividade de
Interacao Humano-Computador. Cobre os 4 cenarios de uso das proto-personas
definidas no documento.

## Como abrir

Abra o arquivo `index.html` em qualquer navegador. Nao precisa de servidor
nem dependencias.

## Cenarios cobertos

1. **Augusto Carlos** (Professor, 42) - "Buscar Sala Agora": filtros rapidos
   (disponibilidade imediata, capacidade 40+, projetor), reserva e QR Code de acesso.
2. **Ana Lima** (Aluna/coord. extensao, 22) - "Novo Agendamento": data/hora,
   categoria, mapa visual do campus (verde = livre / vermelho = ocupada),
   justificativa, comprovante em PDF e convite de calendario.
3. **Marcos Ramos** (Coord. administrativo, 51) - "Painel de Controle" (desktop):
   ocupacao em tempo real, sensores de presenca, alerta de conflito e
   exportacao do Relatorio Mensal de Ociosidade.
4. **Gabriela Ciresi** (Discente de SI, 19) - "Buscar Salas Ocupadas": busca por
   disciplina/codigo e rota de navegacao ate a sala realocada.

## Observacao

Todos os dados sao simulados. O foco e demonstrar os fluxos de navegacao,
nao a implementacao funcional.
